# php-ga-tracking-redirect
> Simple PHP Google Analytic Tracking with custom 301 redirection.

## Usage

```sh
$ composer install
$ php -S localhost:8080 -t ./public
```
- Change your `tracking_id` in `public/index.php` file.
- Open in your browser: `http://localhost:8080?url=http://github.com`
- Check out your Google Analitycs dashboard.
